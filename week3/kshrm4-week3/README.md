## Getting Started

First, install the Vercel AI-SDK
```bash
npm install ai @ai-sdk/react @ai-sdk/openai zod
```
Then run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
